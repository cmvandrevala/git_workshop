\section[GitHub Pages]{GitHub Pages and Jekyll}

\begin{frame}
\frametitle{Pages}
\begin{enumerate}
\item GitHub allow users to have one personal (static) site.
\vspace{1mm}
\item It also allows one site per project.
\vspace{1mm}
\item Jekyll is the site-generator behind GitHub Pages.
\end{enumerate}
\end{frame}
\note[itemize]{
\item There are plenty of templates written in Liquid.
See here \url{http://jekyllrb.com/docs/templates/}. You can also use CSS templates.
}
